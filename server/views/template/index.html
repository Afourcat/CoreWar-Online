What's the CoreWar ?

It's on of the first programming game where Warriors are fighting together in the arena ! Created in 1984 by D.G.Jones and J.K.Dewdney this game let programmer create warriors in RedCode !

How it's work ?

Warriors and Arena are as you certainly know virtuals :-(
The arena is a virtual ram and Warriors are RedCode's written programs. Each program send live signal that mean that it's alive. When only one program send live signal that mean that he win.

How to play ?

You have first to learn RedCode which is a pseudo assembly language. Then use your knowledge to make the best warrior that can defeat all his enemies !
